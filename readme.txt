== Description ==

Qorntix is a lightweight, fast and accessible classic WordPress theme for blogs,
portfolios and small business sites. It ships with a full template hierarchy, a built-in
Customizer powered by the core WordPress Customizer API, widget areas, and clean,
standards-compliant code. It does not require any plugin to work.

Features:

* Full classic template hierarchy (index, single, page, archive, search, 404, comments).
* Selectable left / right / no sidebar layout via the Customizer.
* Custom logo, custom menus and a footer widget area.
* Accessible, keyboard-friendly main navigation with a skip link.
* Featured image, threaded comments and block editor (wide & full alignment) support.
* Translation ready and RTL ready.
* No bundled or required third-party plugins.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
4. Navigate to Appearance > Customize to configure layout and other options.

== Frequently Asked Questions ==

= Does this theme require any plugin? =

No. Qorntix works out of the box without any required or bundled plugins.

= Does the theme support the block editor? =

Yes. It supports wide and full-width alignment, block styles and responsive embeds.
